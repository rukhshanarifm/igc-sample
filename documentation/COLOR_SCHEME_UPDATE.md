# Color Scheme Update - Light Green & Minimalistic

## Summary of Changes

Both `index.html` and `td_loss_dashboard.html` have been updated to use a light green and minimalistic color palette instead of the previous dark green theme.

### Color Palette

**Primary Background**
- Light cream/white: `#f8faf9` → `#f0f5f3` gradient

**Header**
- Light green gradient: `#e8f5e9` → `#c8e6c9`
- Header text: `#1b5e20` (dark green for contrast)

**Shadows (Minimalistic)**
- Reduced from `0 8px 32px rgba(0,0,0,0.1)` to `0 1px 3px rgba(27, 94, 32, 0.05)`
- Creates subtle, clean appearance

**Accent Colors**
- Primary button: `#81c784` → `#66bb6a` (light green gradient)
- Active states: `#66bb6a` (medium light green)
- Card borders: `#a5d6a7` (very light green)

**Text Colors**
- Body text: `#2d3e2f` (dark with slight green tint)
- Primary headings: `#2e7d32` (medium green)
- Secondary text: `#666` (neutral gray)

### Key Updates

1. **Header**: Changed from dark blue/green to light green gradient
2. **Cards & Containers**: Reduced shadow effects for minimalistic look
3. **Buttons**: Updated to light green with subtle gradient
4. **Borders**: Changed from dark to light green tones
5. **Focus States**: Updated input focus colors to match light green theme
6. **Tab Active State**: Changed to light green indicator

### Result

The dashboards now have a:
- ✅ Clean, minimalistic appearance
- ✅ Light green theme throughout
- ✅ Subtle shadows for modern flat design
- ✅ Better visual hierarchy with reduced contrast
- ✅ Professional and modern look
